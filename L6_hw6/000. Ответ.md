## 2. Unit файлы:
```console
[Unit]
Description=PingPong multiplayer game on port 8080
After=network.target

[Service]
Type=simple
ExecStart=/home/ubuntu/WebSocket-Pong-Game/pingpong
WorkingDirectory=/home/ubuntu/WebSocket-Pong-Game
Restart=always
User=ubuntu
Group=ubuntu
StandardOutput=append:/home/ubuntu/logs/go/output.log
StandardError=append:/home/ubuntu/logs/go/output.log

[Install]
WantedBy=multi-user.target
```

```console
[Unit]
Description=RockPapaerScissors
After=network-online.target

[Service]
Type=simple
ExecStart=/home/ubuntu/rock-paper-scissors-k8s/app/venv/bin/python /home/ubuntu/rock-paper-scissors-k8s/app/app.py
WorkingDirectory=/home/ubuntu/rock-paper-scissors-k8s/app/
Environment="PYTHONPATH=/home/ubuntu/rock-paper-scissors-k8s/app/venv/lib/python3.12/site-packages"
Restart=always
User=ubuntu
Group=ubuntu
StandardOutput=append:/home/ubuntu/logs/python/output.log
StandardError=append:/home/ubuntu/logs/python/output.log

[Install]
WantedBy=multi-user.target
```

## 3. Файл конфигруации logrotate:
```console
/home/ubuntu/logs/go/output.log {
daily
rotate 3
size 1M
compress
delaycompress
}
```
